# 工作日志管理 (Work Log)

智能工作日志记录助手。自动识别部门、工时、加班时间、日期等参数，并优化描述内容。

**触发词**：worklog, work-log, 工作日志, 日志, 记录工作

## 核心功能
- 自动识别参数（部门、工时、日期、描述）
- 优化描述内容（简洁专业）
- 智能合并同部门记录（工时累加，描述追加）
- 格式：`- [部门] - [工时] - 描述 - [加班X小时]`

## 使用示例
```
/work-log 智慧人社事业部 1小时 解答数据同步问题
/work-log 昨天 研发部 3h 完成认证模块开发
/work-log @202601.md 技术支持 2小时 解决数据库连接问题
```

## 执行逻辑
1. 提取参数：部门、工时、加班工时、日期、文件路径、描述
2. 优化描述：去除冗余，提炼核心，保留关键信息
3. 智能默认：当前日期、当前月份日志文件
4. 合并记录：同日期同部门自动合并（工时累加，描述追加）
